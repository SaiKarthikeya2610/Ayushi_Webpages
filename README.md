# 🌿 Ayushi Webpages UI

This project contains a set of responsive static webpages designed for **Ayushi**, developed using **HTML** and **CSS** in **VS Code**. The design includes a custom hexagonal green logo with the letter "A".

---

## 📄 Pages Included

- ✅ Sign Up Page  
- ✅ Sign In Page  
- ✅ Contact Us Page
